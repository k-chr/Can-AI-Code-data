Write a program that takes 10 integers from the user, two at a time, and writes them to consecutive elements of an array. After retrieving these numbers, display on the screen one by one:

* their average value, with an accuracy of 4 places, 
* the maximum value, 
* the minimum value.

End all three lines with newline characters.

If the user provides incorrect data, the program should display the message: 'Input data type error' and terminate with error code '1'.

Use exactly one call to the 'scanf' function in the program.
