# Can-AI-Code-data

This repository contains data to support the reproduction of the results presented in the paper:
***"Can AI Code? A Head-to-Head Showdown Between Large Language Models and Student Programmers in C"***

## Repository structure

- tasks - contains text files with task descriptions in English
- reports - contains reports generated by the automated-grading system "Dante"
- codes - contains codes generated by each analysed model in the paper
- tests - contains a test suite extracted from reports for each task.
